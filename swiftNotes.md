# Swift
* for iOS, OS, and watch OS

## Comments
* start with //

## Simple values
* constants
  * use the "let" keyword
  * doesn't have to be known to compile
  * must be assigned a value once
* variables
  * use the "var" keyword

## Type Conversions
* ex. String(value)
* including values in strings
  * "I have \(numFruits) fruits."
